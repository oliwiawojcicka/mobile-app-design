# Virtual Fridge App Design

A UX/UI design project for a mobile application that helps users manage food stored at home.

The application allows users to track available ingredients, quantities, and expiration dates. It also supports recipe discovery, identifies missing ingredients, and helps create shopping lists.

## Main Features

* Food inventory management
* Expiration date tracking
* Barcode scanning and manual product entry
* Recipe suggestions
* Ingredient availability comparison
* Shopping list creation
* Product notifications

## Design Process

The project includes user research, competitor analysis, information architecture, sketches, wireframes, usability testing, design iterations, and a final interactive prototype.
